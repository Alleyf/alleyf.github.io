> 使用hexo的fluid主题搭建的GitHub Pages静态博客，记录个人笔记、随笔等信息。

### ✨ [演示](https://alleyf.github.io/)

## 安装

```sh
npm install
```

## 使用

```sh
hexo cl && g && d
```

## 运行测试

```sh
hexo s
```
